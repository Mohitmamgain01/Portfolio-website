# Personal Portfolio Website

A modern and fully responsive personal portfolio website built using **HTML, CSS, and JavaScript**. The website showcases my education, technical skills, projects, resume, and contact information through a clean and interactive user interface.

## Features

- 👨‍💻 Personal introduction section
- 🙋 About Me section
- 🎓 Education timeline
- 💻 Technical skills showcase
- 🚀 Projects section
- 📄 Resume download section
- 📞 Contact information
- 🌙 Dark/Light mode toggle
- ✨ Smooth scrolling navigation
- 📱 Fully responsive design
- 🎨 Modern UI with animations and hover effects

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

- Home

## How to Run

1. Clone or download this repository.
2. Place your profile image as `profile.jpg` in the project folder.
3. Open `index.html` in any modern web browser.
